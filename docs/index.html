THE VISION

World-class tooling
for a national LLM.

Amália represents a major achievement in open, sovereign AI for the Portuguese language. Amalia 2.0 exists to make that achievement accessible, delightful, and production-ready for every developer.

We are building the missing layer: a beautiful, agent-native, template-driven SDK and CLI that feels like it was made for the modern AI builder — while staying deeply respectful of Portuguese language, culture, and the original AMALIA-LLM vision.

This is not just an SDK. It is infrastructure for Portuguese-speaking builders and the next generation of European AI applications.

GET STARTED IN SECONDS

Quickstart

1. INSTALL
pip install amalia2
# or for development
pip install -e ".[cli,agents]"
2. CREATE A NEW PROJECT
amalia2 new my-portuguese-app --template agent-chatbot
3. START CHATTING
amalia2 chat --model amalia-9b-instruct
